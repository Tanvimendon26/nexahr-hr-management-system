# 🚀 NexaHR – HR Management System

NexaHR is a modern **Full Stack Human Resource Management System (HRMS)** designed to simplify workforce management through role-based access, attendance tracking, leave management, task assignment, and employee administration.

Built with **React, TypeScript, Node.js, Express, and SQLite**, the application provides separate dashboards for **Admin, HR, and Employees**, delivering an intuitive and efficient experience.

---

## ✨ Features

### 🔐 Authentication & Authorization
- Secure role-based login (Admin, HR, Employee)
- Protected routes
- Profile management
- Change password functionality

### 👨‍💼 Employee Management
- Add, edit, and delete employees
- Employee directory
- Department assignment
- Employee profile management

### 🏢 HR Management
- HR account management
- Workforce administration
- Employee monitoring
- HR profile management

### 📅 Attendance Management
- Clock In / Clock Out
- Attendance history
- Working hours calculation
- Attendance statistics
- Daily attendance tracking

### 📝 Leave Management
- Apply for leave
- Approve or reject leave requests
- Leave status tracking
- Leave history

### ✅ Task Management
- Assign tasks to employees
- Track task progress
- Priority management
- Task completion workflow
- Task analytics

### 📊 Dashboard & Analytics
- Role-specific dashboards
- Attendance insights
- Employee statistics
- Task statistics
- Department overview

### 👤 Profile Management
- Update personal information
- Change password
- Role-specific profile pages

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/Tanvimendon26/nexahr-hr-management-system.git
```

Navigate into the project

```bash
cd nexahr-hr-management-system
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Open the application

```
http://localhost:3000
```
